The Computer Vision API gives the developer the tools to process images by Analyzing, Describing, generating Thumbnails, listing Domain Specific Models, Optical Characte Recognition (OCR), recognizing Domain Specific Content, and Tagging. Analyzing provides us with details such as tags and categories based on the objects in the image. Describing attempts to give a generated description of what is happening within the picture. Thumbnails can be generated by identifying a "region of interest" (ROI) within the image to crop out unnecessary elements. Domain Specific Models are based on specialized information about certain categories such as "classic cars" within the bigger category "cars." However, currently the Computer Vision API only supports celebrity recognition and analysis for their Domain-Specific Content. In this guide, I will focus on getting started with the API for analyzing images using pure JavaScript. Below is a link to the official documentation.

Requirements

  • Supported input methods: Raw image binary in the form of an application/octet stream or image URL
  • Supported image formats: JPEG, PNG, GIF, BMP.
  • Image file size: Less than 4MB.
  • Image dimension: Greater than 50 x 50 pixels.
